## API

| 属性      | 说明                                 | 类型                                 | 默认值     |
| --------- | ------------------------------------ | ------------------------------------ | ---------- |
| align     | 对齐方式                             | `small \| middle \| large \| number` | -          |
| direction | 间距方向                             | `vertical \| horizontal`             | horizontal |
| size      | 间距大小                             | `SpaceSIze`                          | small      |
| wrap      | 是否自动换行，仅在 horizontal 时有效 | `boolean`                            | -          |

- SpaceSIze

`'small' | 'middle' | 'large' | number`

### 插槽

| 属性  | 说明     | 类型    | 默认值 |
| ----- | -------- | ------- | ------ |
| split | 设置拆分 | `slots` | -      |
